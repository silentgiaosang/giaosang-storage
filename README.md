
本仓库是基于 STM32 HAL 库开发的个人例程集合，集成度很高，方便复刻

## 📖 例程介绍
### 1. oledmenu
    模块化设计，新增菜单仅需修改结构体数组，无需改动底层驱动；支持 I2C 通信方式，适配主流 128x64 OLED 屏。


### 2.oled_game
  基于Google小恐龙设计，功能完备。

## 🔧 硬件环境
| 硬件模块       | 型号/参数                          |
|----------------|------------------------------------|
| 主控芯片       | STM32F407VET6（兼容其他 F1/F4 系列） |
| OLED 显示屏    | SSD1306 驱动，I2C 通信，128x64 分辨率 |
